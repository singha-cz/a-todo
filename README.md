# a-todo
A simple React To-do list app based on React.js functional components and Context API

## Features
- Full task management - Add/Remove/Edit/Search/Filter/Mark complete
- Progress bar
- Uses LocalStorage
- Keyboard support (Esc, Enter, Tab)
- Soft UI design
- Themeable (via Sass)
